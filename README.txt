= Readme =

jbx_appointment_booking is a TYPO3 extension which allows basic appointment booking.

== Requirements ==

* TYPO3 4.5
* smarty

== Configuration ==

* TypoScript

plugin.tx_jbxappointmentbooking_pi1 {
    gcal_username = your_username
    gcal_password = your_password
}
